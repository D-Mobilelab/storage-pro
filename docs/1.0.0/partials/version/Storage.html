Storage

Description

Storage is the library to save and get data in different ways.

Import & Usage

<script type="text/javascript" src="./bower_components/storage-pro/dist/dist.js">
Now you can use global object Storage and associated methods, described below.
You can start by initializing it with init method.

Methods

  • delete(key, options)

    This method is used to delete a stored value.

    Parameters
    ParamTypeDetails
    keystring

    key to identify value

    optionsObject

    (see param below)

    PropertyTypeDetails
    typestring

    type of storage where delete data, use this only if different from default value defined before in init method

    (default: null)

    Example

    // Storage delete variable from default type storage (defined in init method)
Storage.delete('country');

// Storage delete variable as cookie
Storage.delete('country', {type: 'cookie'});

  • get(key, options)

    This method is used to get a value from the storage.

    Parameters
    ParamTypeDetails
    keystring

    key to identify data

    optionsObject

    (see param below)

    PropertyTypeDetails
    typestring

    type of storage where load data, use this only if different from default value defined before in init method

    (default: null)

    Returns
    *

    saved data

    Example

    // Storage loads variable from default type storage (defined in init method)
Storage.get('country', 'it');

// Storage loads variable as cookie
Storage.get('isLogged', {type: 'cookie'});

  • getMultiple(keys, options)

    This method is used to get multiple values from the storage.

    Parameters
    ParamTypeDetails
    keysArray

    keys to identify data

    optionsObject

    (see param below)

    PropertyTypeDetails
    typestring

    type of storage where load data, use this only if different from default value defined before in init method

    (default: null)

    Returns
    *

    saved datas

    Example

    // Storage loads variables from default type storage (defined in init method)
Storage.getMultiple(['country', 'isLogged']);
// it returns {country: 'it', isLogged: true}

// Storage loads variables as cookie
Storage.getMultiple(['country', 'isLogged'], {type: 'cookie'});
// it returns {country: 'it', isLogged: true}

  • init(options)

    Init storage module.

    Parameters
    ParamTypeDetails
    optionsObject

    (see attributes below)

    PropertyTypeDetails
    typestring

    Define default storage that you want to use.
    It must be one of these options:

    • localstorage
    • cookie
    • jsobject

    (default: 'cookie')

    loggerObject

    any object containing the following methods: debug, log, info, warn, error

    verboseboolean

    if true Storage logs also get methods

    (default: false)

    Example

    Storage.init({
  type: 'cookie',
  logger: console
  verbose: false
});

  • isLocalStorageSupported()

    This method is used to know if the browser support the local storage

    Returns
    boolean

    true if supported, else false

    Example

    if(Storage.isLocalStorageSupported()) {
     // store a localstorage key
 } else {
     alert('local storage is not supported');
 }

  • set(key, value, options)

    This method is used to set a value in the storage.

    Parameters
    ParamTypeDetails
    keystring

    key to identify data

    value*

    data to store

    optionsObject

    (see param below)

    PropertyTypeDetails
    typestring

    type of storage where save data, use this only if different from default value defined before in init method

    (default: null)

    exdaysInteger

    expirations days of saved data (valid only for cookie)

    (default: null)

    exhoursInteger

    expirations hours of saved data (valid only for cookie, if exdays is undefined)

    (default: null)

    exminutesInteger

    expirations minutes of saved data (valid only for cookie, if exdays and exhours are undefined)

    (default: null)

    Example

    // Storage saves variable in default type storage (defined in init method)
Storage.set('country', 'it');

// Storage saves variable as cookie
Storage.set('isLogged', true, {type: 'cookie'});

  • setMultiple(key, options)

    This method is used to set multiple values in the storage.

    Parameters
    ParamTypeDetails
    keyObject

    data to save

    optionsObject

    (see param below)

    PropertyTypeDetails
    typestring

    type of storage where save data, use this only if different from default value defined before in init method

    (default: null)

    Example

    // Storage saves variables in default type storage (defined in init method)
Storage.setMultiple({country: 'it', isLogged: true});

// Storage saves variables as cookies
Storage.setMultiple({country: 'it', isLogged: true}, {type: 'cookie'});
